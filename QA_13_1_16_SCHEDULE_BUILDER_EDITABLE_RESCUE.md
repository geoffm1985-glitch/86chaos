# QA 13.1.16 Schedule Builder Editable Rescue

1. Deploy to staging.
2. Run Emergency Schedule Rescue for July 2026.
3. Open Time Clock & Schedule > Schedule Builder.
4. Go to July 2026.
5. Click an existing rescued shift and delete it.
6. Add a replacement shift for the same employee/date.
7. Refresh. The replacement should still show.
8. Add a new shift to an empty July date.
9. Refresh. The new shift should still show.
10. Publish schedule. Draft count should publish normally.
