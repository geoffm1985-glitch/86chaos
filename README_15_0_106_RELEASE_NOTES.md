# 86 Chaos 15.0.106 Release Notes

## System Administrator Content Rebuild

This release rebuilds the System Administrator work area beyond the previous menu-only pass.

### Changed
- Bumped app version to 15.0.106.
- Added the `system-admin-v106` content workbench layer.
- Converted clicked System Administrator pages into dark, compact, scroll-contained work areas.
- Removed the old pale/white admin card bleed that made retention/legal schedule cards unreadable.
- Added a section map under each System Administrator page heading so related tools are visible without hunting through the left nav.
- Made retention setup, legal schedule, production checklist, and expected jobs readable in compact dark rows.
- Kept the lean status bar instead of adding a bulky dashboard header.
- Kept all existing System Administrator controls, except Branding / Display remains removed from System Administrator and belongs in Settings.

### Not changed
- No Firebase, Vercel, rules, permissions, QuickBooks, push, or backend behavior was changed.
- No customer-facing System Administrator access was added.
