# 86 Chaos 15.0.12 Release Notes

## Menu Intelligence Help Guide

This release adds a proper public Help Center article for Menu Intelligence so managers know exactly how to use the intelligent menu tools.

## Changes

- Added a dedicated **Using Menu Intelligence** Help Center article.
- Added step-by-step instructions for uploading a menu photo/PDF, reviewing AI-detected menu items, matching ingredients to inventory, and approving reviewed menu links.
- Added instructions for editing and deleting Recent Menu Scans.
- Added explanation of how approved Menu Intelligence links power 86 menu-impact alerts.
- Added examples for kitchen shorthand, including burger/patty/product-code inventory names like `BEEF GR PTY`.
- Updated the Administrator Manual with 15.0.12 support and QA guidance.

## Deploy notes

Deploy through GitHub/Vercel. No new Firestore rules, Storage rules, Vercel environment variables, or separate Firebase publishing are required for this build.
